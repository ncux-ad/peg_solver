# Peg Solver

Решатель головоломки с колышками (Peg Solitaire) с использованием гибридных алгоритмов.

## Описание

Программа для решения классической головоломки "Колышки" (Peg Solitaire). Использует комбинацию алгоритмов поиска с эвристиками и обратным решателем для нахождения оптимальных решений.

## Технологии

- Python 3.x
- Алгоритмы поиска (A*, обратный поиск)
- Эвристические функции
- Кэширование решений

## Быстрый старт

1. Запустите решатель:
```bash
python main.py
```

2. Или используйте программный интерфейс:
```python
from solver import hybrid_solver
from visualizer import display_board

board = parse_input("size=7x7 pegs=... empty=D4")
solution = hybrid_solver(board)
print(format_solution(solution))
```

## Основные функции

- Гибридный решатель (прямой + обратный поиск)
- Эвристические функции для оптимизации
- Кэширование решений
- Визуализация доски и решений
- Поддержка различных размеров доски

## Структура проекта

```
peg_solver/

```

